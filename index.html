예시)

  2. 결과 = 배열의 총합 / 배열의길이

        let result = 0;
        const reducer = (previousValue, currentValue) => previousValue + currentValue;
      
        if (arr) {
          result = arr.reduce(reducer) / arr.length;
        }
        return result
      }

        return array.reduce((a, b) => a + b) / array.length;
      }

    Q. 삼각형의 두 변의 길이가 담긴 배열 sides이 매개변수로 주어집니다. 나머지 한 변이 될 수 있는 정수의 개수를 return하도록 solution 함수를 완성해주세요.

    선분 세 개로 삼각형을 만들기 위해서는 다음과 같은 조건을 만족해야 합니다.

    가장 긴 변의 길이는 다른 두 변의 길이의 합보다 작아야 합니다.


    - sides의 원소는 자연수입니다.

    - sides의 길이는 2입니다.

    - 1 ≤ sides의 원소 ≤ 1,000


    예시)

    - sides: [1, 2] = 결과: 1

    - sides: [3, 6] = 결과: 5

    - sides: [11, 7] = 결과: 13

    ■ 내가 작성한 코드

    1. 가장 긴변이 b인 경우
      • b ＜ a + c
      • max = min + c
      • c = max + min
      • max ~ c (가장 긴변의 값부터 max + min값까지)
      • max + min - max // c - max
    2. 나머지 한변이 제일 긴 경우
      • a + b > c > max(a,b)
      • min + max > c > max
      • c = min + max - 1 - max
      • c = min - 1
    3. answer = max + min - max + min - 1
      
function solution(sides) {
    var answer = 0;
    const min = Math.min(...sides)
    const max = Math.max(...sides)

    answer = min * 2 -1

    return answer;
