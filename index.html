Array.slice(begin, end)

- begin : 0을 시작으로 하는 추출 시작점에 대한 인덱스
- end: 추출을 종료할 0 기준 인덱스, end 생략시 배열의 끝까지 추출함
- const res = arr.slice(1, 4); // index 2 ~ 4 까지 추출
- const res2 = arr.slice(2, -1);//index 2 ~ 끝에서 2번째까지 추출
